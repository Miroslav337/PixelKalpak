# Pixel Kalpak

Desktop library management system built with Python and CustomTkinter.

## Features

- Book catalog — add, edit, delete books with authors, genres, and copy counts
- Readers — manage reader profiles, lending history, and notes
- Lending — issue and return books with configurable loan periods
- Dashboard — live stats and charts (books on hand, top genres, monthly issues)
- Export — save data to CSV, TXT, XLSX, or JSON
- Multilingual UI — Russian, English, Kyrgyz
- Light / Dark theme with accent color presets
- Admin panel — multiple admins with role-based access (admin / superadmin)

## Requirements

Python 3.10+

```
pip install -r requirements.txt
```

## Run

```
python main.py
```
